About The Project

Press for GitHub repository

PhishGuard is an application designed to protect users from phishing attacks by analyzing potentially malicious text-based content. The application is currently available in demo form as a Google Chrome web browser extension, with a front end interface created through node.js. On any webpage, the user can open the extension from their hotbar and highlight a specific area on the page which they want analyzed in a manner similar to a snipping tool. Divisions falling within the bounds are scraped for text, which is then processed. Text analysis is performed using dual large language models from Cohere, independently trained on 1000+ unit datasets of email-based and SMS-based phishing samples respectively. Upon completion of analysis, a pop-up will appear over the current window, with options for information on how to proceed and avoid the attack or to close the pop-up and continue browsing.
